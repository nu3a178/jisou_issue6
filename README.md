### テックブログ

Qiitaの技術記事と、MicroCMSのブログ情報を取得して表示するWebページです。
Qiita記事はクリックで該当の記事へ、MicroCMSはクリックでそのまま中身のコンテンツを表示できます。

ヘッダーの画像は都度ランダムに表示されます。

トップページ以外のページはサーバーサイドレンダリングで表示されています。

### 環境変数

.envファイル内に以下の変数を設定してください。

QIITA_APIKEY = Qiitaの設定からAPI keyを発行してください。
DEPLOY_DOMAIN= デプロイ先のドメイン
MICROCMS_API_KEY= MicroCMSの設定から取得してください。
MICROCMS_SERVICE_ID= MicroCMSの「サービス管理」から参照可能です。
